# Project Overview

This project is a Flutter e-commerce application. It allows users to browse products, add them to a shopping cart, and checkout. The application uses Firebase for authentication and as a backend.

# Implemented Features

*   **User Authentication:** Users can register and log in using email and password. Firebase Authentication is used to manage user accounts.
*   **Product Catalog:** Products are displayed in a catalog with their titles, prices, and images. The app uses a `ProductProvider` to manage product data.
*   **Shopping Cart:** Users can add products to a shopping cart. The `CartProvider` manages the items in the cart.
*   **Navigation:** The application has a navigation drawer that allows users to navigate between different screens.
*   **UI/UX:** The application has a modern and clean user interface, with a consistent color scheme and layout.
